# Controle-de-Ganhos-para-Motoristas-de-Aplicativo-Python
Sistema desenvolvido para registrar corridas, calcular ganhos, despesas e lucro final, utilizando listas, dicionários e lógica de programação.
Este projeto é um sistema desenvolvido em Python para auxiliar motoristas de aplicativo no controle de ganhos diários.

O sistema permite registrar corridas, calcular o valor total recebido, despesas e lucro final, além de fornecer uma visão simples do desempenho financeiro.

Foi desenvolvido com foco em prática de lógica de programação, estruturas de dados e organização de informações.
